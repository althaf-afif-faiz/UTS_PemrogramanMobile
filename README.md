## KoffNet UI

KoffNet UI adalah desain antarmuka aplikasi pemesanan kopi modern dengan tampilan elegan dan pengalaman pengguna yang intuitif.  
Project ini dibuat sebagai prototype untuk aplikasi mobile (Android/iOS).

## Fitur Tampilan

1. **Splash & Welcome Screen**  
   - Animasi logo dan transisi ke halaman utama.

2. **Menu Screen**  
   - Daftar menu kopi dan makanan.  
   - Gaya tampilan grid minimalis.

3. **Order & Checkout**  
   - Detail pesanan.  
   - Pilihan metode pembayaran dan konfirmasi order.

4. **Rating & History**  
   - Pengguna dapat memberi ulasan.  
   - Riwayat transaksi tersimpan di layar khusus.
